# 🚀 NexaWave Agency

NexaWave Agency is a modern and dynamic landing page designed for a software and digital agency.  
This project highlights a clean UI, smooth animations, and a strong visual hierarchy to create a professional and engaging user experience.

The design focuses on showcasing services, portfolio, and brand identity in a sleek and impactful way, making it ideal for tech agencies and startups.
